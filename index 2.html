The score

From one instruction to a shipped, verified report.

Five movements. You set the tempo once — the fleet plays it out and hands you back a single, checkable result.

01◆ load-brain

Load the Brain

Your context, memory, and past work become a browsable layer every agent can read — with provenance on every fact, not a black box.

02✳ set-the-score

Set the score

Compose the run on the canvas: pick the orchestrator model, wire in the tools, skills, and prompts each agent is allowed to touch.

03⑃ fan-out

Fan out

One instruction spawns a fleet. Each agent takes a branch, a PR, or a Linear ticket and works in parallel — you watch it happen on the graph.

04◉ watch-live

Watch live

Agent-aware preview, ambient cost, and an attention rail that surfaces only what needs a human — before the fleet stalls waiting on you.

05✓ the-report

Get the report

Every run converges into one scheduled report: what shipped, what it cost, what was verified, and what's queued next.
