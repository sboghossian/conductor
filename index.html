The score

From a prompt and some files to a shipped, verified report.

Five movements. You set the tempo once — the fleet plays it out and hands you back a single, checkable result.

01" prompt + files

Drop the ask

Start with a prompt and a dataroom — attach the brief, the Figma, the spreadsheet. Everything you drop becomes context the fleet can read.

02◆ load-brain

Load the Brain

Your files meet your memory and past work in one browsable layer — with provenance on every fact, not a black box.

03⑃ fan-out

Fan out

One instruction spawns a fleet. Each agent takes a branch, a PR, or a Linear ticket and works in parallel — every one a session you can open.

04◉ watch-live

Watch live

Agent-aware preview, ambient cost, and an attention rail that surfaces only what needs a human — before the fleet stalls waiting on you.

05✓ the-report

Get the report

Every run converges into one scheduled report: what shipped, what it cost, what was verified, and what's queued next.

The wall

Not a chat box. A cockpit.

Everything a chat window makes you do by hand — attention, cost, memory, trust, replay — Conductor makes visible and operable.

Any model. Any agent. Swap anytime.

Pick an agent type per session — reviewer, builder, migrator, researcher — and choose the model that drives it. Change models or rotate API keys mid-run, no restart.
